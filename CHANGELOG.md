# 更新日志

本文档记录平台主要功能变化。线上功能没有生效时，请先确认服务器已经执行：

```bash
cd /opt/csppractice
git pull
pm2 restart csppractice
```

然后在浏览器中强制刷新页面：

- Windows：`Ctrl + F5`
- macOS：`Cmd + Shift + R`

## 2026-05-07

### 550ff0d - 练习成绩展示优化

- 学生提交客观题后，答题卡显示醒目的成绩卡。
- 显示百分比大分数，例如 `82%`。
- 显示原始分数，例如 `41 / 50 分`。
- 显示正确题数、错误题数、总题数。
- 练习记录中的客观题成绩改为彩色分数徽章。
- 班级作业中的客观题成绩改为更明显的百分比分数标签。

### 172a334 - CSP 试卷取消等级要求

- CSP-J 初赛、CSP-S 初赛、CSP-J/S 初赛试卷不再需要设置等级。
- 首页只有选择 `GESP` 类型时才显示等级筛选。
- 教师创建试卷时：
  - 选择 `GESP`：显示并保存等级。
  - 选择 CSP 类型：隐藏等级，并保存为 `level: null`。
- 学生学习中心的等级进度只统计 GESP 试卷。

### b7f88b1 - 题目内容支持 Markdown

- 题干、解析、错题本、编程题题面支持 Markdown 渲染。
- 支持代码块，例如：

````markdown
```cpp
int s = 0;
for (int i = 1; i <= 5; i++) {
    if (i % 2 == 1) s += i;
}
cout << s;
```
````

- 支持行内代码、加粗、斜体、列表、简单表格。
- 修复 CSP-J/S 样卷中的题干显示问题。

### 49b6a34 - 平台更名并增加 CSP 初赛方向

- 平台更名为：`初赛考级练习平台`。
- 移除左上角图片 Logo，改为文字品牌。
- 首页增加考试类型筛选：
  - GESP
  - CSP-J 初赛
  - CSP-S 初赛
  - CSP-J/S 初赛
- 教师可视化建卷中增加“考试类型”字段。
- 默认添加 `CSP-J/S 初赛综合样卷`。

### dca54f3 - 增加 Ubuntu 22.04 部署文档

- 新增部署文档：`docs/ubuntu-22.04-deploy.md`。
- 包含 Node.js、PM2、Nginx、服务器更新、首次上线注意事项。

### 9d1442c - 学生流程和可视化建卷优化

- 新增学生学习中心。
- 新增待完成作业、已完成作业、等级进度、错题本。
- 教师题库管理从纯 JSON 改为可视化表单建卷。
- 支持添加单选题、判断题、编程题。
- 保留高级 JSON 导入/导出。

### 75ffd7b - 初始平台版本

- 试卷列表。
- 登录和注册。
- 客观题自动判分。
- C++ 编程题本机 `g++` 评测。
- 练习记录。
- 班级、作业、教师管理台、管理员用户管理的基础能力。
