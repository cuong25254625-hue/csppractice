# 更新日志

本文档记录平台主要功能变化。线上功能没有生效时，请先确认服务器已经执行：

```bash
cd /opt/csppractice
git pull
npm install
pm2 restart csppractice
```

```bash
cd /opt/csppractice
git pull origin main
npm install --omit=dev
pm2 restart csppractice
pm2 save
```

然后在浏览器中强制刷新页面：

- Windows：`Ctrl + F5`
- macOS：`Cmd + Shift + R`

## 2026-05-09

### 本次调整 - 教师可查看并发布共享试卷

- 教师管理后台恢复显示全部试卷，可查看其他教师或管理员创建的试卷内容。
- 教师可将其他账号创建的试卷发布为自己班级的作业，便于共用题库资源。
- 非本人创建的试卷在教师端显示为只读，只能查看和导出，不能修改、删除、隐藏/显示或通过 Word 导入覆盖。

### 本次更新 - 管理台内置网站使用说明

- 管理台顶部新增“使用说明”入口，点击后跳转到站内说明页。
- 使用说明覆盖快速开始、账号角色、试卷设置、Word 导入导出、班级作业、学生管理、学生端使用、服务器更新备份和常见问题。
- 说明页增加左侧章节目录，便于管理员和教师快速跳转到需要查看的配置说明。

## 2026-05-08

### 本次更新 - Word 试卷导入导出

- 管理台试卷列表新增 Word 导出功能，支持导出选中的试卷或当前筛选结果为 `.docx` 文件。
- 导出的 Word 文件使用可编辑模板格式，包含试卷信息、题型、题目 ID、分值、题干、选项、答案和解析。
- 管理台新增 Word 导入功能，可上传按模板编辑后的 `.docx` 文件并写回题库，同 ID 试卷会更新，不存在的试卷会新建。
- 导入导出适合教师线下整理、复制和批量录入试题，减少在网页表单中逐题录入的时间。

### 本次更新 - 班级作业与学生管理增强

- 学生端班级页面调整为先进入具体班级，再查看该班级发布的试卷作业，避免所有班级作业混在一起。
- 学生端班级列表不再显示邀请码，邀请码仅教师或管理员在管理台可见。
- 管理台班级页将“发布作业”提升到学情区域上方，并在当前班级学情打开时默认选中该班级。
- 教师和管理员可在班级学情中批量添加学生到班级，支持添加选中学生或一键添加全部可选学生。
- 管理员创建或批量导入学生账号时，可为学生绑定所属老师；教师批量加学生时只看到自己名下学生，管理员可看到全部学生。
- 账号菜单新增“修改密码”，登录用户可输入当前密码后自行更新新密码。

### 本次更新 - 管理试卷支持类型筛选与名称搜索

- 管理试卷列表新增考试类型下拉筛选，可按 GESP、CSP-J、CSP-S 等类型过滤试卷。
- 新增试卷名称搜索框，输入关键词实时过滤匹配的试卷。
- 筛选生效时标题栏显示"匹配数 / 总数"，并提供"清除筛选"按钮一键恢复全部显示。
- 搜索输入使用防抖（300ms），避免频繁重渲染。

### 本次更新 - 提交确认与未作答提醒

- 提交客观题时弹出确认对话框，显示已作答/未作答题目数量，防止误提交。
- 存在未作答题目时，对话框显示醒目警告，明确提示"未作答将计 0 分"。
- 全部已作答时，显示绿色确认提示。
- 用户可点击"返回继续作答"取消提交，回到试卷继续答题。

### 本次更新 - 修复判断题未作答误判正确

- 修复阅读程序/完善程序中的判断子题：用户不选任何选项直接提交时，后端误将 `undefined` 默认转为 `false`，若正确答案恰好是 `false` 则错误判为正确。
- `normalizeAnswerValue` 增加未作答检测，未作答时返回 `undefined` 而非默认值。
- `answersEqual` 增加未作答检测，未作答时直接返回 `false`（不得分）。

### 本次更新 - 试卷管理和 CSP 题型增强

- 建卷页面增加题号导航，可点击题号快速跳转到对应题目。
- 题号导航改为编辑器内悬浮导航条，滚动到下方建题时也能快速跳转。
- 建卷页面增加批量修改分值工具，可批量设置全部题、客观题或复合题子题分值。
- 优化建题区域的题目卡片分隔、编号和强调线，降低题目之间看串行的概率。
- 错题本改为按试卷分组，并使用折叠式错题卡片展示，先看摘要，需要时再展开选项和解析。
- 管理试卷列表增加 `隐藏/显示` 按钮，隐藏后的试卷不会出现在学生端试卷列表、学习中心作业和班级作业中。
- 隐藏/显示重新设计为 SQLite 运行时可见性状态，不再修改题库 JSON，也不再重写整套试卷内容。
- 管理试卷支持多选后批量隐藏或批量显示。
- 前端静态资源增加版本参数，降低浏览器缓存旧隐藏接口导致失败的概率。
- 创建新试卷时默认 ID 从 `0001` 起自动取下一个可用编号，避免一直使用相同 ID。
- 建卷页面在题目列表底部也提供添加题目工具栏，不需要每次回到顶部添加题目。
- 新增多选题题型，支持多选作答、自动判分和错题展示。
- 新增 `阅读程序题` 和 `完善程序题` 两类复合题，支持填写题面、程序代码，并在题目内配置判断、单选、多选子题。

### 本次更新 - 暂停编程题提交并迁移运行时存储

- 编程题提交默认关闭，后端 `/api/submit-code` 会返回暂未开放提示，前端不再显示代码提交入口。
- 管理台默认隐藏 `添加编程题` 按钮，待运行沙箱配置完成后再重新开放。
- 运行时数据从 `data/db.json` 迁移到 SQLite `data/runtime.sqlite`，并继续保留 `db.json` 作为首次初始化导入源。

### 本次更新 - 彻底关闭 MathJax 辅助探索弹窗

- 进一步关闭 MathJax 的语义增强、公式探索、语音和盲文相关开关。
- 页面加载前清理旧的 `MathJax-Menu-Settings` 浏览器缓存，避免历史设置重新启用公式探索模式。
- 增加 `mjx-help` 隐藏兜底样式，防止点击公式后出现小感叹号和英文说明弹窗。

### 本次更新 - 试卷编辑跳转和后台入口修复

- 修复右上角账号下拉菜单中教师/管理员看不到 `管理后台` 的显示问题。
- `管理试卷` 和 `创建/修改试卷` 改为独立页面状态，点击创建或修改后跳转到编辑页。

### 本次更新 - 试卷管理拆分和代码行号

- 管理试卷支持单套修改、单套删除和多选批量删除。

- 管理员可以在用户管理中上传 Excel 批量创建账号。

- 上传后可选择批量创建为学生账号或教师账号。
- 导入后显示成功数量，并列出重复账号、格式错误等跳过原因。

### 本次更新 - 管理后台和答题卡修复

- 修复作答试卷时点击答题卡题号会跳回首页的问题。
- 管理后台整理为 `试卷题库`、`班级学情`、`系统设置` 三个页签。
- 班级学情入口改为醒目的 `查看学情` 按钮。
- 班级报告空状态增加明确提示，教师和管理员更容易找到查看位置。

### 本次更新 - 提交与账号菜单体验优化

- 答题卡按钮文案从 `提交客观题` 改为 `提交`。
- 提交后的 `客观题成绩` 改为 `成绩`。
- 插入代码改为弹窗体验：粘贴长代码后可以先预览，再插入到题目描述。
- 右上角账号改为下拉菜单，集中放置 `个人学习`、`管理后台` 和 `退出登录`。

- 修复客观题成绩为 `100%` 时百分号或文字在窄区域换行的问题。
- 可视化建题增加 `插入代码`、`插入公式`、`插入图片` 快捷按钮。

- 管理员可以在后台新增、删除考试类型。
- 内置考试类型保留：`GESP`、`CSP-J 初赛`、`CSP-S 初赛`。
- 试卷可视化建卷中的考试类型改为读取后台配置，不再写死固定类型。
- 题干、解析、编程题题面在建卷表单中明确提示支持 Markdown 代码块。
